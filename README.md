    <h2> 📌Principais Tecnologias Utilizadas
        </h2>
    
<hr>
    
<p> O desenvolvimento web pode ser dividido em três camadas principais:

</p>
<ol>
        <li>
            <b>Frontend</b> - Responsável pela interface visual. Tecnologias principais:
            <ul>
            <li>HTML (estrutura)</li>
            <li>CSS (estilo)</li>
            <li>JavaScript (interatividade)</li>
            
</ul>

</li>
        
<li>
            <b> Backend</b> - Responsável pelo funcionamento interno do site. Principais tecnologias:
            <ul>
            <li>Node.js</li>
            <li>Python (Django, Flask)</li>
            <li>PHP</li>
            
</ul>

</li>
        
 <li>
            <b> Banco de Dados</b> -Onde as informações são armazenadas. Alguns exemplos:
            <ul>

<li>MySQL </li>            
              <li>PostgreSQL</li>
              <li>MongoDB</li>

</ul>
        </li>

   </ol>
    <h2> 🔥Tendências o Desenvolvimento Web</h2>
    <hr>
    <p> Algumas tendências estão moldando o futuro da web:
    </p>
    <ul>
        <li> <b>Aplicações Web Progressivas (PWA)</b> -  Sites que funcionam como apps nativos.</li>
        <li><b>Inteligência Artificial</b> - Personalização de experiências com IA. </li>
        <li><b>Web 3.0 e Blockchain</b> - Tecnologias descentralizadas para segurança e privacidade. </li>
    </ul>

        </li>

   </li>
